# 一叶归真项目说明

## Design Context

### Users
**Primary Audience**: 樊文花品牌管理层（C级高管、品牌总监、战略决策者）
**Context**: 正式商务会议前后查看，需要快速理解合作价值和可行性
**Job to be Done**: 评估一叶归真作为合作伙伴的专业实力和这次联名合作的战略价值

### Brand Personality
**Voice**: 专业、创新、国际化
**Tone**: 自信但不傲慢，现代但有底蕴，精致但不繁复
**3-Word Personality**: 专业、现代、可信
**Emotional Goals**: 让管理层感受到信任感、专业度和合作兴奋感

### Aesthetic Direction
**Visual Tone**: 现代国际化 - 类似苹果/谷歌/Stripe 的简约现代风格
**Theme**: 浅色主题，突出内容清晰度和专业感
**References**: 
- Stripe Dashboard (简洁专业的数据呈现)
- Apple Keynote 演示文稿 (清晰的视觉层级)
- Notion 界面 (现代简洁的信息架构)
**Anti-References**: 
- 传统商务PPT模板
- 过度装饰的中式设计
- 深色"科技感"界面

### Design Principles
1. **信息层级优先**: 关键数据和价值主张要立即可见，次要信息可以渐进展示
2. **国际化现代感**: 使用国际化的设计语言，体现公司的全球视野和现代化水平  
3. **完全独立的视觉**: 不受一叶归真主品牌深色+金色风格约束，为B2B场景定制
4. **交互驱动**: 通过精心设计的交互增强用户体验和信息传达效果
5. **移动优先**: 考虑到管理层可能用手机/平板查看，确保移动端体验优秀
